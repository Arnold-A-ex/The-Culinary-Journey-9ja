# The Culinary Journey 9ja Show 🍳🇳🇬

The official web platform for **The Culinary Journey 9ja Show**, a nationwide cooking competition designed to discover Nigeria’s most talented and creative cooks. This platform serves as the digital hub for contestant registration, culinary training, and brand storytelling, focusing on the rich heritage of the Niger Delta region.

## ✨ Features

- **Premium Visual Identity:** Custom-themed UI using Culinary Gold, Jet Black, and Mangrove Green.
- **Interactive Training Portal:** A 10-module accordion-based curriculum for aspiring chefs.
- **WhatsApp Integration:** A functional contact system that formats user inquiries directly into WhatsApp messages.
- **Competition Roadmap:** Clear details on eligibility, judging criteria, and the 2026 prize pool (₦1M + Brand New Car).
- **Responsive Design:** Optimized for mobile, tablet, and desktop using Tailwind CSS v4.

## 🛠️ Tech Stack

- **Framework:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router Dom v6](https://reactrouter.com/)
- **Icons:** Lucide-React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/Arnold-A-ex/The-Culinary-Journey-9ja.git](https://github.com/Arnold-A-ex/The-Culinary-Journey-9ja.git)
   cd The-Culinary-Journey-9ja

 * Install dependencies
   npm install

 * Start the development server
   npm run dev
